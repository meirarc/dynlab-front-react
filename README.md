# dynlab-front-react

Create a React app

`npx create-react-app dynlab`

Starting a project

`cd dynlab`

`npm start`

Buld a project

`npm run build`

Install Amplify & Configure ** every time when open gitpod **

`npm install -g @aws-amplify/cli`

`amplify configure`

Install amplify on the app

`npm install aws-amplify`

`npm install aws-amplify-react`
